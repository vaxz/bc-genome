geneAccessionComp()

geneAccessionComp(\bcGen\MainBundle\Entity\Gene  $gene1, \bcGen\MainBundle\Entity\Gene  $gene2) : integer

This method compares 2 Genes objects according to their accessionNum.

Only the first four characters are used. The choosen order is : BCIN > B0510 > BC1G > BCT4 > BOFU

Parameters

\bcGen\MainBundle\Entity\Gene $gene1
\bcGen\MainBundle\Entity\Gene $gene2

Returns

integer —

0 if the 2 Gene objects are the same 1 if gene1 is superior to gene2 -1 if gene1 is inferior to gene2
