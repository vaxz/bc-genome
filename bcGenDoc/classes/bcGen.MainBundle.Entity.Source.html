\bcGen\MainBundle\Entity

\bcGen\MainBundle\EntitySource

Source class allows the creation of Source object and correponds to a table in the database : bc_gen_db.

It's related to documents (publication, inner communication) which give information on other objects (gene, genome...). Publication and Innercommnication classes inherit from Source class.
